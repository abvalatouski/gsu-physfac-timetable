## Usage

```
Usage: ptt.EXE [-w FILEPATH | -W FILEPATH] [-b FILEPATH | -B FILEPATH]
               [-t FILEPATH | -T FILEPATH]
  Downloads timetable from "old.gsu.by/physfac" and serializes it into JSON

Available options:
  -w FILEPATH              Where to place weeks "above and under the line"
  -W FILEPATH              `-w` producing formatted output
  -b FILEPATH              Where to place bell rings table
  -B FILEPATH              `-b` producing formatted output
  -t FILEPATH              Where to place Excel timetable
  -T FILEPATH              `-t` producing formatted output
  -h,--help                Show this help text
```
